如何用hexo和github建立一个blog

发表于 2020-12-21 更新于 2020-12-29 分类于 Web

1. 目的

本文的目的是在github上建立一个blog;使用typora来编辑markdown格式的内容;使用了hexo工具来生成html文件.

2. github

github page(https://pages.github.com/) 是用来给你的project写website的一个功能,我们用它来存放我们的blog.

2.1 注册github

这个不用多说了,直接去 www.github.com 注册一个.

2.2 新增一个github page的repository

左上角的New,新增一个新的repository

repository name输入<你的github用户名>.github.io

点开这个repository,在setting里面可以找到github pages的选项

在这里可以选择Theme

安装git和hexo

我使用的是windows平台,blog的内容放在d:\blog下.先创建好这个目录.

安装git

下载windows下的git,并安装https://git-scm.com/download/win

配置git

我们使用ssh key的方式链接github,所以要做一些配置.

在d:\blog里点邮件,选择git bash here,打开git bash

配置用户名和邮箱

先生成key pairs

在github里的setting->”SSH and GPG keys”,右上角的”New SSH key”, 把刚才生成的公钥帖进去.

安装node.js

下载node.js,并安装. https://nodejs.org/en/

用下面的命令改成国内的阿里的镜像.

安装hexo

安装hexo的插件,使得用typora插入的图片,生成的html也可以使用

修改D:\blogs\node_modules\hexo-asset-image\index.js
